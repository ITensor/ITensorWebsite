Official Julia Codes

These are Julia packages developed by or in collaboration with the ITensor team. We support these codes at the same level as the core ITensor library itself.


PastaQ is a package for working with quantum hardware and circuits, including tomography, optimization, and simulations.

Developers: Giacomo Torlai, Matt Fishman


ITensorGPU

Package enabling ITensor code to be run on NVIDIA GPUs using the CUDA.jl Julia library backend.

Developers: Katharine Hyatt, Matt Fishman


ITensorGaussianMPS

Tools for transforming Gaussian (non-interacting) states of fermions into matrix product states (MPS).

Developers: Matt Fishman

ITensorVisualizationBase

Tools for visualizing tensor networks with multiple graphical backend options.

Developers: Matt Fishman


ITensorInfiniteMPS

Package for optimizing infinite matrix product states (iMPS) with the VUMPS algorithm and working with iMPS.

Developers: Matt Fishman



Official C++ Codes

These are C++ codes developed by or in collaboration with the ITensor team. We support these codes at the same level as the core ITensor library itself.


TDVP

The official implementation of the time-dependent variational principle (TDVP) algorithm for the C++ version of ITensor.

Developers: Mingru Yang


Community Codes

These are codes based on ITensor by other members of our community. While we cannot guarantee or provide technical support for these codes, we still welcome discussion of them in our support forum and encourage users to try out and contribute to these community codes.


DMRGPy

DMRGPy is a Python library built on top of ITensor to compute quasi-one-dimensional spin chains and fermionic systems using matrix product states with DMRG as implemented in ITensor. Most of the computations can be performed both with DMRG and exact diagonalization for small systems, which allows one to benchmark the results.

Developers: Jose Lado
