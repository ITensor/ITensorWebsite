Jul 17, 2023


We will soon be removing the function format from the itensor:: namespace in the next version of the C++ ITensor library. The reason is to remove a conflict with the std::format function, available in C++20.

If your code uses the format function, please call tfm::format from now on instead if you'd like to call exactly the same function (tfm:: stands for "tinyformat", the underlying library supplying this function). Or else update your code to use std::format instead.
