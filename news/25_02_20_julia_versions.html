Incompatibilities with Minor Julia Versions

February 20, 2025

Please note that there are issues installing the latest version of ITensors.jl (ITensors.jl v0.8) in older versions of Julia v1.10 and v1.11, the only official releases of Julia that we support right now. Please use the latest official releases of Julia v1.10 and v1.11, which are v1.10.8 and v1.11.3 at the time this was written. See https://julialang.org/downloads for up-to-date information about the latest releases of Julia and recommended ways of installing Julia. Using the latests Julia releases is good practice anyway, and juliaup is a good way to easily stay up-to-date with the latest releases (again, see https://julialang.org/downloads for details).
